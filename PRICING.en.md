# GCC-Evo Pricing

See also: [LAYER_STRUCTURE.md](LAYER_STRUCTURE.md) for the authoritative v5.330 free/paid 8-layer directory split.

## Canonical Tier Boundary

- **Community**: UI + L0 + L1 + L2 + L3
- **Evolve ($29/month)**: Community + commercial enhancement packs in `paid/l0-l3`
- **Pro ($79/month)**: Evolve + paid core `L4 + L5`
- **Enterprise ($500+/month)**: Pro + `DA` + private deployment

## Tier Table

| Tier | Price | Included | Best For |
|------|-------|----------|----------|
| Community | Free Forever | UI + L0 + L1 + L2 + L3 | Personal, academic, <$1M revenue |
| Evolve | $29/month | + paid/l0-l3 enhancement packs | Teams needing stronger governance, math, and acceptance workflows |
| Pro | $79/month | + L4 + L5 | Teams needing decision evolution and closed-loop orchestration |
| Enterprise | $500+/month | + DA + private deployment | Large funds and enterprise deployments |

## Notes

- `L4` is not part of the canonical free tier.
- `L5` is not part of the canonical free tier.
- `DA` is not part of the canonical free tier.
- Release-facing messaging should use the `5 Free + 3 Paid` layer model.
- Legacy modules may still exist for backward compatibility, but the canonical commercial boundary is defined by `LAYER_STRUCTURE.md`.
