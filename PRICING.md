# GCC-Evo 定价

另见: [LAYER_STRUCTURE.md](LAYER_STRUCTURE.md)，该文件是 v5.325 免费/付费 8 层目录边界的权威说明。

## 权威层级边界

- **Community**: UI + L0 Phase 1 + 基础 L1/L2/L3/L5
- **Evolve ($29/月)**: Community + L0 Phase 2-4
- **Pro ($79/月)**: Evolve + 完整 L1/L2/L3
- **Enterprise ($500+/月)**: Pro + L4 + 高级 L5 + DA + 私有部署

## 层级表

| 层级 | 价格 | 包含内容 | 适用对象 |
|------|------|----------|----------|
| Community | 永久免费 | UI + L0 Phase 1 + 基础 L1/L2/L3/L5 | 个人、学术、营收低于 $1M 的团队 |
| Evolve | $29/月 | + L0 Phase 2-4 | 需要验证数据质量与数学建模的团队 |
| Pro | $79/月 | + 完整 L1/L2/L3 | 需要完整记忆、检索与蒸馏能力的团队 |
| Enterprise | $500+/月 | + L4 + 高级 L5 + DA + 私有部署 | 大型资金团队与企业部署 |

## 说明

- `L4` 不属于免费 canonical tier。
- `DA` 不属于免费 canonical tier。
- `L0` 必须明确区分免费 `Phase 1` 与付费 `Phase 2-4`。
- 仓库中保留的 legacy 模块仅用于兼容，不定义商业边界；商业边界以 `LAYER_STRUCTURE.md` 为准。
