# GCC-Evo 定价

另见: [LAYER_STRUCTURE.md](LAYER_STRUCTURE.md)，该文件是 v5.400 免费/付费 8 层目录边界的权威说明。

## 权威层级边界

- **Community**: UI + L0 + L1 + L2 + L3
- **Evolve ($29/月)**: Community + `paid/l0-l3` 商业增强包
- **Pro ($79/月)**: Evolve + 付费核心 `L4 + L5`
- **Enterprise ($500+/月)**: Pro + `DA` + 私有部署

## 层级表

| 层级 | 价格 | 包含内容 | 适用对象 |
|------|------|----------|----------|
| Community | 永久免费 | UI + L0 + L1 + L2 + L3 | 个人、学术、营收低于 $1M 的团队 |
| Evolve | $29/月 | + `paid/l0-l3` 商业增强包 | 需要更强治理、数学建模、验收流程的团队 |
| Pro | $79/月 | + `L4 + L5` | 需要决策进化与闭环编排的团队 |
| Enterprise | $500+/月 | + `DA` + 私有部署 | 大型资金团队与企业部署 |

## 说明

- `L4` 不属于免费 canonical tier。
- `L5` 不属于免费 canonical tier。
- `DA` 不属于免费 canonical tier。
- 发布口径必须使用 `5 层免费 + 3 层付费` 模型。
- 仓库中保留的 legacy 模块仅用于兼容，不定义商业边界；商业边界以 `LAYER_STRUCTURE.md` 为准。
