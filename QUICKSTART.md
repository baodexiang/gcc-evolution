# QUICKSTART - gcc-evo v5.400

本手册用于快速启动 `opensource` 版本的 `gcc-evo`。

## 发布口径

- `免费基础层`: `UI + L0 + L1 + L2 + L3`
- `付费核心层`: `L4 + L5 + DA`
- `paid/l0-l3`: 商业增强包

## 1. 安装

```bash
pip install gcc-evo
gcc-evo version
```

## 2. 初始化项目

```bash
gcc-evo init
```

## 3. 配置 L0

```bash
gcc-evo setup KEY-001
gcc-evo l0 scaffold
gcc-evo l0 check
```
