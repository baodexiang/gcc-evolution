## What Is gcc-evo?

`gcc-evo` helps an LLM agent improve across sessions instead of starting from zero every time. It provides:

- persistent memory
- hybrid retrieval
- experience distillation
- governed task loops
- dashboard visibility

## Product Boundary

The public product boundary is:

- `5 free foundation layers`: `UI`, `L0`, `L1`, `L2`, `L3`
- `3 paid core layers`: `L4`, `L5`, `DA`
- `paid/l0-l3` are enhancement packs, not extra top-level layers
