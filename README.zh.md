# gcc-evo v5.440

`gcc-evo` 是一个面向 LLM Agent 的自进化引擎，提供记忆、检索、蒸馏、治理和闭环编排能力。

## 对外口径

- `5 层免费基础层`：`UI`、`L0`、`L1`、`L2`、`L3`
- `3 层付费核心层`：`L4`、`L5`、`DA`
- `paid/l0-l3` 属于商业增强包，不额外计入顶层层数

## 安装

```bash
pip install gcc-evo
gcc-evo version
```
