# Legacy Compatibility

This directory documents compatibility-only packages that still exist at the package root:

- `L0_setup`
- `L1_memory`
- `L2_retrieval`
- `L3_distillation`
- `L4_decision`
- `L5_orchestration`
- `observer`
- `direction_anchor`
- `enterprise`

These names remain in the repository to avoid breaking existing imports.
They do not define the commercial boundary.

Canonical boundary:

- free -> `gcc_evolution/free/`
- paid -> `gcc_evolution/paid/`
